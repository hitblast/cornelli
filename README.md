## Usage

Get started by putting your thoughts away for a certain duration:

```bash
# unlocks after 20 days, 1 hour, 2 minutes and 1 second
nelli keep "You are doing great, keep going..." -d 20d1h2m1s
```

As time goes, you'll eventually be able to check back using the following command:

```bash
nelli mailbox
```

Hopefully you'll enjoy what comes next! :3

## Compiling

```bash
cargo build --release --locked
```
